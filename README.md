# homebrew-brewfile
My Brewfile to do a clean install on my MacBook

Install [Homebrew](https://brew.sh/)

Drink from that [Brewfile](https://github.com/jeffpaul/homebrew-brewfile/blob/master/Brewfile) goodness

- Downloads
- Music
- ZSUZSI

- Hangouts > Chrome
- h: meet.google.com/

Manually install:
- Pokemon TGC
- Sonos
- Sophos
